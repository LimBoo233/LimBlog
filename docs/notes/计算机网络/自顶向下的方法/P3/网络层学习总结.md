# 网络层总结 (Network Layer: Summary)

这张幻灯片总结了我们在网络层控制平面部分学习到的主要内容：

- 网络控制平面的两种主要方法 (Approaches to network control plane)：
    - 每个路由器独立控制 (Per-router control - 传统方式)。
    - 逻辑上集中的控制 (Logically centralized control - 软件定义网络SDN)。
- 传统路由算法 (Traditional routing algorithms)：
    - 我们学习了链路状态 (Link State) 和距离向量 (Distance Vector) 算法的原理。
- SDN控制器 (SDN controllers)：
    - 在实践中的实现 (Implementation in practice)：
        - ODL (OpenDaylight)。
        - ONOS。
- 互联网控制报文协议 (Internet Control Message Protocol - ICMP)。
- 网络管理 (Network management)：
    - 包括SNMP, NETCONF/YANG等协议和技术。
