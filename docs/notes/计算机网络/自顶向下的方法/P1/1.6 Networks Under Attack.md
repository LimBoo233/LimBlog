# DDos
在网络世界里，DDoS 攻击是指攻击者利用多台被控制的计算机（通常是被恶意软件感染的“僵尸网络”），向目标服务器或网站发送大量的恶意请求或数据流量，使其不堪重负，最终导致目标服务中断，无法响应正常的合法用户请求。

# Lines of defense
1. authentication: 通过身份认证
2. confidentiality: 通过加密 via encryption
3. integrity check: digital signature prevent/detect modification
4. access restriction: password-protected VPNs
5. firewalls: 只允许特定的数据进入 filter packets
