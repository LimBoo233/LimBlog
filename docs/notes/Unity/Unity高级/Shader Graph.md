# Shader Graph

## Intro & Start

Shader Graph 无需手写代码，通过创建和连接节点来构建 Shader，并且能即时看到修改结果。适合只想做**常用简单效果，不想深入图形学太多**的使用者。

**Shader 本质上是运行在 GPU 上的小程序**，用来决定一个物体最终怎么显示。

比如：
- 这个物体是什么颜色？  
- 表面粗糙还是光滑？  
- 是否发光？  
- 是否透明？  
- 贴图怎么采样？  
- 边缘要不要溶解？  
- 顶点要不要晃动？

一个 Shader Graph 大概由三部分组成：

### ① 输入

比如：

颜色  
贴图  
法线贴图  
噪声  
时间  
UV  
世界坐标  
顶点法线  
自定义参数
