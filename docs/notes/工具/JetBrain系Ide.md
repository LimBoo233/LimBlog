## 最 cooooool 的快捷键

- 全知的搜索：`Double Shift`
- 无敌的插入：`Alt + Insert` (Windows) / `Cmd + N` (Mac)
- 万用的重构：`Alt + Enter`

其他常用的：
- 打开文件树：`Alt + 1`
- 文件树搜索：打开文件数直接输入
- 重命名：`Shift + F6`
- 复制上一行：`Ctrl + D`
- 复制/删除/剪切当前行：`Ctrl + C/Y/X`
- qurick doc：`Ctrl + Q`
- 查看参数：`Ctrl + P`
